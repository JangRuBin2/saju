# 컨텐츠 확장 계획: 유명인 사주 비교 및 엔터테인먼트 기능

## 1. 배경 및 시장 조사

### 1.1 기존 서비스 현황

이미 다수의 한국 사주 서비스가 유명인 관련 컨텐츠를 제공 중:

| 서비스 | 주요 기능 | 특징 |
|--------|----------|------|
| 타이트사주 (sajutight.me) | 연예인 궁합 테스트, AI 점수 산출 | BTS/BLACKPINK 등 K-pop 아이돌 포함, 다차원 점수 |
| 닮은사주찾기 (alikesaju.com) | 나와 닮은 유명인 사주 찾기 | 유사도 인덱스 산출, 무료 |
| 청월당 (aifortunedoctor.com) | 유명인 사주 분석 갤러리 | 재물운/연애운/직업운 편집 기사 형태 |
| 데일리운세 (unse.daily.co.kr) | 연예인 궁합 | 사진 포함 DB, 카테고리 분류 |
| 포스텔러 (forceteller.com) | 유명인 사주 필터 | 860만 누적 사용자, 연 100억+ 매출 |

### 1.2 인기 컨텐츠 유형 (트래픽 순)

1. **연예인 궁합 테스트** - 사용자 생년월일 + 연예인 선택 -> 궁합 점수
2. **나와 닮은 유명인 사주 찾기** - 사주 유사도 기반 매칭
3. **유명인 사주 분석 에디토리얼** - 재물운/직업운 등 장문 분석
4. **재물운 비교** - 나 vs 유명인 재물운 비교
5. **나는 연예인 사주일까?** - 도화살, 식상 등 연예인 사주 특징 체크

### 1.3 핵심 타겟 고객

- MZ세대 여성 (전체 사용자의 75-83%)
- 재물운과 궁합이 가장 높은 트래픽

## 2. 법적 검토 결과

### 2.1 유명인(연예인) 사주 분석 - 법적 리스크

| 기능 | 한국법 리스크 | 국제법 리스크 | 판단 |
|------|-------------|-------------|------|
| 유명인 사주 분석 (긍정적 내용) | 중간 | 낮음 | 면책조항 + 긍정적 프레이밍으로 진행 가능 |
| 유명인 사주 분석 (부정적 내용) | **높음** | 중간-높음 | 자동 필터링으로 부정적 예측 차단 필요 |
| 유명인 궁합 (사용자 vs 연예인) | **높음** | 높음 | 기존 서비스들이 운영 중이나 리스크 존재 |
| 유명인 재물운 비교 | 중간 | 낮음-중간 | 사용자 중심 프레이밍으로 진행 가능 |
| 닮은 유명인 찾기 | 낮음 | 낮음 | 안전 - 유사도 매칭은 리스크 낮음 |

### 2.2 주요 법률 리스크 상세

**퍼블리시티권 (인격권 기반)**
- 한국에 별도 법률은 없으나, 판례로 인정 (서울고법 2000나42061)
- 유명인의 성명/초상을 상업적으로 이용하면 민사 불법행위 해당 가능
- 대응: 유명인 사진 미사용, 이름만 텍스트로 사용, 사용자가 직접 입력하는 구조

**명예훼손 (형법 307조)**
- 한국은 **사실 적시도 명예훼손** 가능 (세계적으로 매우 엄격)
- 정보통신망법 70조: 온라인 허위사실 명예훼손 시 최대 징역 7년
- 2026년 탈덕수용소 사건: 징역 2년 집행유예 + 2.1억 추징금
- 대응: 부정적 예측 자동 필터링, 긍정/중립 내용만 노출

**개인정보보호법 (PIPA)**
- 생년월일은 개인정보에 해당
- 유명인의 공개된 생년월일은 회색지대 (공개된 개인정보)
- 대응: 사용자가 직접 데이터 입력하는 구조로 책임 전환

**K-pop 소속사 리스크**
- SM, JYP, HYBE 등은 아티스트 상업적 이용에 매우 적극적으로 법적 대응
- 기존 서비스(타이트사주 등)가 운영 중이나, 소속사 대응은 서비스 규모에 비례
- 대응: 초기에는 글로벌 유명인 위주, K-pop은 사용자 입력 방식으로 제한

### 2.3 만화 캐릭터 - 법적 검토 결과

| 캐릭터 유형 | 법적 리스크 | 사유 |
|------------|-----------|------|
| 현역 프랜차이즈 (나루토, 원피스 등) | **높음** | 상표권 등록됨, 일본 IP 홀더 적극 단속 |
| 클래식 캐릭터 (아톰 등) | 중간 | 저작권 보호 중, 집행 강도 가변적 |
| 퍼블릭 도메인 (손오공, 그리스 신화) | **낮음** | 저작권 없음 |
| 자체 제작 캐릭터 | **없음** | 완전 소유권 |
| 역사적 인물 (사망 50년+) | **낮음** | 인격권 소멸 |

**결론: 만화 캐릭터 사용은 연예인보다 오히려 법적 리스크가 높음** (저작권/상표권 침해). 기존 사주 서비스 중 만화 캐릭터를 사용하는 곳은 발견되지 않음.

### 2.4 안전한 대안 (리스크 낮음 -> 높음)

1. **자체 아키타입 캐릭터** - 사주 유형별 오리지널 캐릭터 (리스크 없음)
2. **역사적 인물** - 세종대왕, 이순신, 레오나르도 다빈치, 나폴레옹 등 (리스크 낮음)
3. **신화/민담 인물** - 손오공, 홍길동, 그리스 신화 등 (리스크 낮음)
4. **퍼블릭 도메인 문학 캐릭터** - 셜록 홈즈, 삼국지 인물 등 (리스크 낮음)
5. **익명화된 유명인 참조** - "1971년생 유명 기술 기업가" (리스크 낮음-중간)
6. **실명 유명인** - 면책조항 + 긍정적 내용 한정 (리스크 중간)

## 3. 컨텐츠 추가 계획

### Phase 1: 안전 컨텐츠 (법적 리스크 최소)

#### 1-1. 사주 아키타입 시스템

일간(日干) 기반으로 10가지 아키타입 캐릭터를 자체 제작하여 사용.

```
예시 아키타입:
- 甲 (갑목): "숲의 리더" - 성장과 도전의 사주
- 丙 (병화): "태양의 표현자" - 열정과 창의의 사주
- 庚 (경금): "강철의 결단자" - 의지와 실행의 사주
- 壬 (임수): "바다의 전략가" - 지혜와 포용의 사주
```

**API 설계:**
```
POST /api/v1/archetype
Body: { year, month, day, hour?, gender }
Response: {
  archetype: { name, element, description, strengths, challenges },
  saju_data: { ... },
  wealth_tendency: "...",
  career_tendency: "..."
}
```

#### 1-2. 역사적 인물과 나의 사주 비교

사망 50년 이상 경과한 역사적 인물 DB를 구축하여 비교 기능 제공.

```
대상 인물 후보:
- 한국: 세종대왕, 이순신, 신사임당, 허준, 이황, 장영실
- 중국: 제갈량, 조조, 유비, 관우 (삼국지)
- 서양: 레오나르도 다빈치, 나폴레옹, 아인슈타인, 링컨
- 일본: 오다 노부나가, 도쿠가와 이에야스
```

**API 설계:**
```
POST /api/v1/compare/historical
Body: { year, month, day, hour?, gender, historical_figure_id }
Response: {
  user_saju: { ... },
  figure_saju: { ... },
  comparison: {
    shared_elements: [...],
    day_master_compatibility: "...",
    wealth_pattern_comparison: "...",
    career_pattern_comparison: "..."
  }
}
```

#### 1-3. "나는 OO 사주일까?" 체크

특정 직업/성향의 사주 특징을 체크하는 엔터테인먼트 기능.

```
체크 유형:
- 연예인 사주 (도화살, 식상 강함)
- CEO 사주 (편재 강함, 겁재 있음)
- 학자 사주 (인성 강함, 식상 있음)
- 예술가 사주 (식상 강함, 화 기운)
- 리더 사주 (비견 강함, 관성 있음)
```

**API 설계:**
```
POST /api/v1/archetype/check
Body: { year, month, day, hour?, gender, check_type: "ceo" | "artist" | ... }
Response: {
  match_score: 78,
  matching_factors: [...],
  description: "..."
}
```

### Phase 2: 유명인 컨텐츠 (면책조항 + 안전장치 필수)

#### 2-1. 나와 닮은 유명인 사주 찾기

사용자 사주와 유사한 유명인을 매칭. 가장 리스크가 낮은 유명인 활용 방식.

```
구현 방식:
- 유명인 생년월일 DB 구축 (공개된 정보만)
- 일간, 오행 비율, 십성 구조 기반 유사도 산출
- 유사도 상위 3명 노출
- 유명인 사진은 미사용 (텍스트만)
```

**API 설계:**
```
POST /api/v1/similar-celebrity
Body: { year, month, day, hour?, gender }
Response: {
  matches: [
    { name, birth_year, similarity_score, shared_traits: [...] },
    ...
  ],
  disclaimer: "..."
}
```

#### 2-2. 유명인 재물운 비교

사용자 재물운과 유명인 재물운을 비교. 사용자 중심 프레이밍.

```
안전 장치:
- 유명인 재물운은 긍정적/중립적 내용만 노출
- "당신의 재물운 특성" 중심으로 프레이밍
- 유명인은 참조 기준점으로만 활용
- LLM 출력에 부정적 예측 필터링 적용
```

**API 설계:**
```
POST /api/v1/compare/wealth
Body: { year, month, day, hour?, gender, celebrity_id }
Response: {
  user_wealth: { pattern, strengths, advice },
  celebrity_wealth: { pattern, strengths },  // 긍정적 내용만
  comparison: { similarities, differences, insight }
}
```

#### 2-3. 유명인 궁합 (높은 리스크 - 신중 진행)

기존 서비스(타이트사주, 데일리운세)가 운영 중이지만 법적 리스크 가장 높음.

```
리스크 최소화 전략:
- "사주 에너지 조화도"로 명칭 변경 (궁합의 연애적 뉘앙스 회피)
- 다차원 점수: 소통 조화, 에너지 조화, 협업 조화, 종합 조화
- 연애/로맨스 프레이밍 완전 배제
- 면책조항 강제 노출
- K-pop 아이돌은 사용자 직접 입력 방식으로만 지원 (DB 미포함)
- 부정적 결과도 긍정적으로 리프레이밍 ("성장의 기회")
```

**API 설계:**
```
POST /api/v1/harmony
Body: {
  user: { year, month, day, hour?, gender },
  target: { name?, year, month, day, hour?, gender }  // 사용자 직접 입력
}
Response: {
  harmony_scores: {
    communication: 82,
    energy: 75,
    collaboration: 88,
    overall: 81
  },
  analysis: "...",
  disclaimer: "..."
}
```

### Phase 3: 프리미엄 컨텐츠 (유료화 연계)

#### 3-1. 사주 유형별 재물 전략 리포트

일간 + 격국 기반 맞춤 재물 전략 제공 (LLM 해석).

```
리포트 구성:
- 타고난 재물 패턴 분석
- 강점 활용 전략
- 주의할 시기 (대운/세운 기반)
- 역사적 인물 중 유사 패턴 사례
```

#### 3-2. 월간/연간 운세 + 유명인 트렌드

해당 월/연도의 운세를 유명인 트렌드와 연결.

```
예시:
"2026년 병오년, 火 기운이 강한 해입니다.
당신의 일간 壬水는 올해 재성이 강해지는 시기...
역사적으로 壬水 일간은 화 기운이 강한 해에 [분석]..."
```

## 4. 필수 법적 안전장치

### 4.1 면책조항 (모든 유명인 관련 기능에 필수)

```
본 서비스의 사주 분석 결과는 전통 역학에 기반한 참고 정보로,
오락 및 교육 목적으로만 제공됩니다.
실제 운명, 재물운, 건강 등을 예측하거나 보장하지 않습니다.
특정 인물에 대한 분석은 공개된 생년월일 정보에 기반하며,
해당 인물의 견해를 대변하지 않습니다.
```

### 4.2 기술적 안전장치

| 안전장치 | 설명 |
|---------|------|
| 부정적 예측 필터 | LLM 출력에서 건강/사망/이혼/파산 등 부정적 예측 자동 차단 |
| 유명인 사진 미사용 | 텍스트(이름)만 사용, 초상권 리스크 제거 |
| 사용자 직접 입력 | K-pop 아이돌 등 민감 대상은 DB 미포함, 사용자가 직접 생년월일 입력 |
| 삭제 요청 대응 | 유명인/소속사 삭제 요청 시 즉시 대응하는 프로세스 구축 |
| Rate Limiting | 유명인 관련 API 호출 제한 (스크래핑 방지) |
| 콘텐츠 모니터링 | LLM 출력 로깅 및 주기적 검토 |

### 4.3 이용약관 필수 조항

- 서비스는 오락 목적이며 실제 운명 예측이 아님을 명시
- 유명인 관련 기능에서 사용자가 입력한 데이터에 대한 책임은 사용자에게 있음
- 저작권/초상권 침해 신고 접수 및 처리 절차 (DMCA 유사)
- 개인정보 처리방침에 유명인 공개 데이터 활용 근거 명시

## 5. 기술 구현 우선순위

### 구현 순서 (리스크 낮음 -> 높음)

```
Phase 1 (즉시 착수 가능)
  1-1. 사주 아키타입 시스템          -- 법적 리스크 없음
  1-2. 역사적 인물 비교              -- 법적 리스크 낮음
  1-3. "나는 OO 사주일까?" 체크      -- 법적 리스크 없음

Phase 2 (면책조항/안전장치 구축 후)
  2-1. 닮은 유명인 찾기              -- 법적 리스크 낮음
  2-2. 재물운 비교                   -- 법적 리스크 중간
  2-3. 에너지 조화도 (궁합 대체)      -- 법적 리스크 중간-높음

Phase 3 (유료화 연계)
  3-1. 재물 전략 리포트              -- 법적 리스크 없음 (사용자 본인)
```

## 6. 최종 권고사항

### 진행 권고

1. **만화 캐릭터 사용은 비권고** - 저작권/상표권 리스크가 연예인보다 높고, 기존 시장에서도 사용 사례 없음
2. **Phase 1부터 시작** - 자체 아키타입 + 역사적 인물로 법적 리스크 없이 컨텐츠 차별화 가능
3. **유명인 기능은 Phase 2에서 신중 진행** - 면책조항/안전장치 완비 후 단계적 확대
4. **궁합 기능은 "에너지 조화도"로 리브랜딩** - 연애적 뉘앙스 제거로 리스크 완화
5. **법률 자문 필수** - Phase 2 착수 전 엔터테인먼트/IP 전문 변호사 자문 권고
