# 반려견 사주 컨텐츠 기획

## 1. 컨텐츠 개요

### 1.1 컨셉

"나와 반려견의 운명적 인연" - 반려견의 사주를 분석하고, 보호자와의 궁합/조화를 해석하는 컨텐츠.

### 1.2 핵심 과제: 반려견 출생 시간 미상 문제

반려견은 정확한 출생 시간을 알기 매우 어려움:
- 유기견/입양견: 출생 연도조차 추정인 경우 다수
- 펫샵/브리더: 출생 날짜는 알지만 시간은 모름
- 직접 출산: 시간까지 아는 경우는 극소수

**대응 전략: 3단계 정밀도 시스템**

| 단계 | 알고 있는 정보 | 분석 가능 범위 | 비고 |
|------|--------------|--------------|------|
| Level 1 | 출생 연도만 (추정 포함) | 연주(年柱) 기반 기본 성향 | 유기견/입양견 대상 |
| Level 2 | 출생 연월일 | 연주+월주+일주 (3주 분석) | 대부분의 반려견 |
| Level 3 | 출생 연월일시 | 사주 전체 (4주 분석) | 직접 출산 목격 시 |

대부분의 사용자는 Level 2에 해당하므로, **3주(연주/월주/일주) 기반 분석을 기본으로 설계**.
시주(時柱) 없이도 충분히 의미 있는 해석이 가능하도록 프롬프트를 구성.

### 1.3 추가 입력: 반려견 고유 정보

시주 부재를 보완하기 위해 반려견 특성 정보를 추가 수집:

| 입력 항목 | 필수 여부 | 용도 |
|----------|----------|------|
| 출생 연도 | 필수 | 연주 계산 |
| 출생 월 | 선택 | 월주 계산 |
| 출생 일 | 선택 | 일주 계산 |
| 출생 시간 | 선택 | 시주 계산 (알 경우만) |
| 성별 | 필수 | 대운 방향 결정 |
| 견종 | 선택 | LLM 해석 보조 (성격 특성 참고) |
| 크기 (소/중/대) | 선택 | LLM 해석 보조 |
| 입양일 | 선택 | 보호자와의 인연 시점 분석 |

## 2. 세부 컨텐츠 구성

### 2.1 반려견 기본 사주 분석

**컨텐츠명:** "우리 아이 타고난 기질"

분석 내용:
- 오행 기반 기질 유형 (활발형/온순형/독립형/사교형/예민형)
- 타고난 건강 체질 (오행 균형 기반 취약 부위 경향)
- 행동 성향 (에너지 레벨, 사회성, 학습 스타일)
- 올해의 컨디션 (세운 기반)

**출력 예시:**
```
[기질 유형: 활발한 모험가 (木 기운 강함)]

2019년 기해(己亥)년에 태어난 우리 아이는...
- 타고난 기질: 호기심이 많고 에너지가 넘치는 탐험가 타입
- 건강 경향: 근골격계가 튼튼하나 소화기 쪽 관리 필요
- 사회성: 다른 강아지와 잘 어울리며 새로운 환경 적응이 빠름
- 학습: 보상 기반 훈련에 잘 반응, 반복보다 다양한 자극 선호
```

### 2.2 보호자-반려견 궁합 분석

**컨텐츠명:** "우리의 운명적 인연"

분석 내용:
- 보호자-반려견 오행 조화도 (상생/상극 관계)
- 궁합 점수 (소통/에너지/생활리듬/유대감)
- 서로에게 미치는 영향 (보호자가 반려견에게, 반려견이 보호자에게)
- 함께하면 좋은 활동 추천
- 주의할 점 (오행 상극 기반)

**핵심 원칙:** 부정적 결과 최소화 - 어떤 궁합이든 긍정적 조언 중심으로 프레이밍.
"상극이라 안 맞는다"가 아니라 "이런 부분을 신경 쓰면 더 좋아진다"로 해석.

### 2.3 반려견 올해의 운세

**컨텐츠명:** "우리 아이 2026년 운세"

분석 내용:
- 올해 건강 운세 (계절별 주의 시기)
- 올해 사회성 운세 (다른 동물/사람과의 관계)
- 보호자와의 유대감 변화
- 월별 컨디션 흐름 (12개월 간단 요약)
- 특별히 좋은 달 / 주의할 달

### 2.4 다견 가정 조화 분석

**컨텐츠명:** "우리 집 댕댕이들 사이"

다견(2마리 이상) 가정을 위한 반려견 간 궁합:
- 반려견 A-B 오행 조화도
- 서열/리더십 경향 분석
- 함께 있을 때 시너지/충돌 포인트
- 새 가족 맞이 시 적응 조언

### 2.5 반려견 입양 적기 분석

**컨텐츠명:** "새 가족을 맞이할 최적의 시기"

입양을 고려 중인 사용자 대상:
- 보호자 사주 기반 올해 반려동물 인연 운세
- 입양하기 좋은 달/시기 추천 (월운 기반)
- 나와 잘 맞는 반려견 오행 유형 추천
- 피해야 할 시기 및 이유

## 4. 프롬프트 설계 원칙

### 4.1 시주 부재 시 프롬프트 처리

```
시스템 프롬프트에 포함:
"이 반려동물의 출생 시간은 알 수 없으므로 시주(時柱)를 제외한
연주, 월주, 일주 3주만으로 분석합니다.
시주 부재로 인한 한계를 언급하되, 3주만으로도 충분히 의미 있는
분석이 가능함을 강조하세요.
'시간을 모르기 때문에 정확하지 않다'는 식의 부정적 표현은 피하세요."
```

### 4.2 반려동물 맞춤 해석 톤

```
- 의인화를 적절히 활용하되 과도하지 않게
- 수의학적 조언이 아님을 명시 (건강 관련 내용 시)
- 모든 결과는 긍정적/중립적 프레이밍
- "우리 아이", "반려견" 등 친근한 호칭 사용
- 견종별 특성은 참고 정보로만 활용 (사주 해석의 보조)
```

### 4.3 면책조항

```
본 분석은 전통 역학에 기반한 재미있는 참고 정보입니다.
반려동물의 건강 문제는 반드시 수의사와 상담하세요.
행동 교정이 필요한 경우 전문 훈련사의 도움을 받으시기 바랍니다.
```

| 컨텐츠 | 판매가 (VAT 포함) | 모델 | API 비용 | 비고 |
|---------|-------------------|------|----------|------|
| 반려견 기본 사주 | 무료 (간단) / 3,300원 (상세) | Haiku / Sonnet | 8원 / 46원 | 무료 버전으로 유입 |
| 보호자-반려견 궁합 | 3,300원 | Sonnet | 60원 | 핵심 유료 컨텐츠 |
| 반려견 연간 운세 | 3,300원 | Sonnet | 46원 | 연초 수요 집중 |
| 다견 가정 조화 | 3,300원 | Sonnet | 60원 | 다견 가정 타겟 |
| 입양 적기 분석 | 3,300원 | Sonnet | 46원 | 입양 예정자 타겟 |

### 시장성 근거

- 한국 반려동물 양육 가구: 602만 가구 (2024년 기준)
- 반려견 관련 월 평균 지출: 15~20만원 (사료, 의료, 용품)
- 반려견 사주/궁합은 기존 사주 서비스에서 거의 다루지 않는 블루오션
- SNS 바이럴 잠재력 높음 ("우리 강아지 사주가 이래요" 공유 욕구)

## 6. 기술 구현 사항

### 6.1 엔진 레이어 변경

- 기존 `SajuCalculator`에 시주 없는 3주 계산 모드 추가 (이미 time_pillar: None 지원)
- 3주 기반 오행 카운트: element_counts 합계 6 (4주일 때 8)
- 대운 계산: 시주 없어도 연주+일간+성별로 대운 방향 결정 가능

### 6.2 모델 레이어 변경

```python
class PetBirthInput(BaseModel):
    name: str | None = None
    year: int = Field(..., ge=1990, le=2030)
    month: int | None = Field(None, ge=1, le=12)
    day: int | None = Field(None, ge=1, le=31)
    hour: int | None = Field(None, ge=0, le=23)
    gender: Gender
    breed: str | None = None
    size: Literal["small", "medium", "large"] | None = None

class PetReadingRequest(BaseModel):
    pet: PetBirthInput
    language: Language = Language.KO

class PetCompatibilityRequest(BaseModel):
    owner: BirthInput
    pet: PetBirthInput
    language: Language = Language.KO
```

### 6.3 서비스 레이어

- `PetService`: 반려견 전용 서비스 (SajuService 위임 + 반려견 맞춤 프롬프트)
- 월/일 미입력 시 연주만으로 간소화된 분석 수행
- 견종 정보는 프롬프트 컨텍스트로만 전달 (엔진 계산에 영향 없음)

## 7. 구현 우선순위

```
Phase 1 (우선):
  1. 반려견 기본 사주 분석 (무료 간단 + 유료 상세)
  2. 보호자-반려견 궁합

Phase 2 (확장):
  3. 반려견 연간 운세
  4. 입양 적기 분석

Phase 3 (다견 가정):
  5. 다견 가정 조화 분석
```
