## 2. 비즈니스 모델

### 2.1 모델 선택: 건당 결제 + 월간 구독 하이브리드

점신(무료+광고)과 포스텔러(코인제)의 장점을 결합하되, 월간 구독 옵션을 추가.
연간 구독은 제외 (구독 피로도 + 결제 허들 고려).

```
[무료 컨텐츠]          --> 유저 확보
  |
[건당 결제]            --> 첫 전환 허들 낮춤 (1,000원~)
  |
[월간 구독]            --> LTV 극대화 (월 4,900원)
  |
[프리미엄 리포트]       --> 고ARPU 컨텐츠 (8,000~10,000원)
```

#### 토스 페이먼츠 수수료

| 항목 | 비율 | 비고 |
|------|------|------|
| VAT (부가가치세) | 10% | 판매가에 포함, 공급가의 10% 납부 |
| PG 수수료 (토스) | ~3.3% | 카드 결제 기준 |

**실수령 계산 공식:**
```
판매가 (VAT 포함) = 소비자 지불 금액
공급가 = 판매가 / 1.1
VAT 납부액 = 판매가 - 공급가
PG 수수료 = 판매가 x 3.3%
실수령 = 공급가 - PG 수수료
```

#### 구독자 1인당 월 수익 분석

```

```

#### 가격 근거

1. **건당 가격 대비 압도적 할인**: 일일 운세 1,100원 x 30일 = 33,000원 -> 4,900원 (85% 할인 효과)

3. **헬로우봇(26,000원) 대비 1/5 가격**: 공격적 진입 전략
4. **MZ세대 월 지출 가능액(6,700원) 내**: 부담 없는 구독
5. **API 비용 대비 충분한 마진**: 구독자당 월 560원 비용, 마진율 87%

## 4. 구독제 상세 설계 (월간만)

### 4.1 구독 상품 구성

```
"오늘의 운세 패스" - 월간 구독만 운영

월간 구독: 4,900원/월 (VAT 포함)

포함 컨텐츠:
  - 매일: 오늘의 운세 (상세) + 행운 컬러/숫자/방위
  - 매주: 주간 운세

비포함 (건당 결제 필요):
  - 종합 사주 감정
  - 특화 운세 (연애/우정/결혼)
  - 신살 분석
```

연간 구독을 제외한 이유:
- 요즘 사용자들은 연간 결제를 꺼림 (구독 피로도)
- 월간 구독이 진입 허들이 낮고 해지가 자유로움
- 4,900원/월은 부담 없이 시작 가능한 가격대

### 4.2 구독자 전용 기능

| 기능 | 설명 |
|------|------|
| Push 알림 | 매일 아침 오늘의 운세 알림 |

| 건당 결제 10% 할인 | 구독자 대상 추가 결제 할인 |
